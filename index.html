Let's Start to learning git

  • git init - это создание репозитори (хранилище данных)
  • Git config user.name and user.email - это конфигурация(создание) имени и Эмейла
  • Git add . || git add index.html - (Отслеживаемая зона) Точка добавляет все файлы в зону сохранения. *Название* Добавляет только 1 файл в зону сохранения
  • Git status - это информация об отслеживаемой зоне
  • Git commit - это создание точки сохранения отслеживаемой зоны только после *Git add*
  • Gitk - открытие и просмотр репозитория
  • Git commit -m'сообщение' это создание точки сохранения отслеживаемой зоны с комментарием
